# Varix Dots

A OpenSUSE Hyperland configuation (rice). Featuring dynamic system-wide theme changing.

|-------------------|------------|
| Compositor 🪟| [hyprland](https://github.com/hyprwm/Hyprland)  |
| Terminal 🖥️       | Alacritty     |
| Shell 🐚          | Fish |
| Fetch 🖼️          | macchina|
| File Manager 📁   | Thunar / [ranger](https://github.com/ranger/ranger)      |
| Editor 📝         | Micro / VSCodium    |
| Browser 🌐        | [firefox](https://github.com/topics/firefox-browser) |
| Bar 📊            | [waybar](https://github.com/Alexays/Waybar)      |
| Launcher 🚀       | [rofi](https://github.com/davatorium/rofi)          |
| Color Theme 🎨    | [pywal](https://github.com/dylanaraps/pywal) |
| Lockscreen 🔒     | [hyprlock](https://github.com/hyprwm/hyprlock)  |
| Login Menu 🚪     | [sddm](https://github.com/sddm/sddm)          |
| Music Player 🎵   | [ncspot](https://github.com/hrkfdn/ncspot) / Spotify   |
| Dotfile Manager | Toml Bombadil |
